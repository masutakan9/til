# qmk_firmware

### qmk_firmwareとは

- 自作キーボードで使われることが多いマイコンのソフトウエア。
- pro microというavrマイコン＋USB付きのボードを利用する例をよく見かける。
- ATMega328pというUSBをサポートしていないavrで使っている例も見るが、これはUSBをソフト的に扱うV-USBというライブラリを利用しているらしい。
